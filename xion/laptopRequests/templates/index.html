About this search:

You can use this to search through the records of laptop requests. Give all of the fields you want to search followed by a ":" and then what you want to look for. Separate each of these search parameters by a space.

Searchable fields:

  • status - The current status of the request (new, started, testing, ready, archive)
  • date - The date the request was submitted
  • name - The submitter's name
  • email - The submitter's email
  • notify - Whether email notification was selected
  • readyby - The 'Requested by' date
  • nlaptops - The number of laptops requested
  • experimentName - The experiment name
  • experimentNumber - The experiment number (NOTE: many are missing!)
  • location - Experiment location
  • backpack - Backpack selected
  • typeX - Experiment type
  • typeL - Laptop type (MAC, PC)
  • owner - Owner: PASSCAL/FA
  • ninstr - The number of instruments
  • diskspace - Diskspace requested
  • media - Other media needed: Usbfire, DVD, Tape
  • software - Additional software requested
  • notes - Submitter notes
  • item - Item typer (hdd, laptop, etc...)
  • model - Equipment model
  • barcode - PASSCAL/FA barcodes
  • serial - Equipment serial numbers (NOTE: many are missing!)
  • specs - Laptop preparer spec notes
  • comments - Laptop preparer equipment notes

Examples:

  • To search for experiments requested on 2009-01-15, try this:

    date:2009-01-15

  • To search for requests with associated equipment owned by PASSCAl with 54 in the barcode from 2009, try this:

    date:2009 barcode:54 owner:PASSCAL
